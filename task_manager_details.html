تطبيق إدارة المهام (Task Manager)

مشروع متكامل مبني لتبسيط وتنظيم العمليات اليومية وإدارة فرق العمل.

عرض حي للمشروع العودة لمحفظة الأعمال

نظرة عامة

تم تصميم هذا التطبيق ليكون حلًا مركزيًا لإدارة المهام وتتبع التقدم. يتيح النظام للمستخدمين إنشاء مهام جديدة، تحديد أولوياتها، تعيينها لأعضاء الفريق، ومتابعة حالة الإنجاز في الوقت الفعلي. يركز التطبيق على **الأداء العالي** و**سهولة الاستخدام**.

التقنيات المستخدمة

PHP MySQL JavaScript/jQuery HTML5 & CSS3 [اسم إطار عمل PHP المستخدم مثل: Laravel أو CodeIgniter]

المميزات الرئيسية

  • **إدارة المستخدمين والأدوار:** نظام تسجيل دخول آمن يدعم أدوار مختلفة (مدير، موظف).
  • **إنشاء وتتبع المهام:** إضافة المهام مع تحديد الموعد النهائي والأولوية والحالة.
  • **تصفية وبحث متقدم:** إمكانية البحث الفوري وتصفية المهام حسب التاريخ أو الأولوية أو المستخدم المُعيّن.
  • **واجهة مستخدم متجاوبة:** تصميم متوافق مع كافة الأجهزة (هواتف، حواسيب لوحية، حواسيب مكتبية).
  • **تكامل قاعدة البيانات:** استخدام MySQL لضمان سرعة استرجاع البيانات ومعالجة حجم كبير من المهام.
